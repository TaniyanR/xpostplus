# セキュリティ運用

## 秘密情報

API認証情報はAPP_KEYで暗号化されます。APP_KEYはリポジトリへ保存せず、サーバーの環境変数として管理してください。

## 初回管理者作成

管理者が存在しない場合でも、XPOSTPLUS_ALLOW_INSTALL=1が設定されていなければアカウントは作成されません。初回設定後は必ず0へ戻してください。

## バックアップ

DBとAPP_KEYの両方をバックアップしてください。APP_KEYを失うと暗号化済みAPI設定は復号できません。

## 公開範囲

Webサーバーのドキュメントルートはpublic/だけに設定し、app/、config/、storage/、.envを外部公開しないでください。
